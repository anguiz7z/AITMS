Security posture

ATMS is offline-first by design. Every cycle of development goes through a security gate: no clones of untrusted repos, no pip install from research, no shell-out for ingest, no eval on user content, no network calls during analysis. WeasyPrint is opt-in; the bundled @media print stylesheet means the browser can produce audit-ready PDFs without any C-library install.
